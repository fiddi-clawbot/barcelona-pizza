# Barcelona Pizza Blog - Build Plan

## Phase 1: Project Setup
- Initialize Jekyll site structure
- Configure GitHub Pages deployment
- Set up Git repo

## Phase 2: Design & Layout
**Color Scheme (Dark Modern):**
- Background: #0f1419 (deep dark)
- Text: #e8eaed (light gray)
- Accent: #ff6b35 (vibrant orange) or #ff9500
- Secondary: #2d3748 (dark gray)
- Links/CTA: #ffa500 or #ff6b35

**Components:**
- Responsive header/nav
- Hero section on home
- Card grid layout (responsive)
- Detail page template
- Footer

**Mobile First Approach:**
- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3 columns

## Phase 3: Research Barcelona Pizzerias
Need to find:
1. Award-winning pizzerias (Michelin, local awards)
2. Featured in Timeout Magazine Barcelona
3. Trending/Instagram-famous spots
4. Information: address, website, specialty pizza, story

**Candidates to research:**
- Pizzeria Beddia (if expanded to Barcelona)
- Local award winners
- Timeout featured venues
- Trending spots on social media

## Phase 4: Build Core Pages
1. **Home Page**: Hero + introductory text + grid of 5 pizzerias
2. **Pizzeria Detail Pages**: 
   - Hero image
   - Pizzeria name, location, cuisine type
   - Description/story
   - Signature pizza with description
   - 2-3 additional images
   - Contact: website, address, phone
   - Google Maps embed/link
   - Awards/recognition

## Phase 5: Implementation Order
1. Create Jekyll structure
2. Design & CSS (dark theme)
3. Build home page template
4. Build detail page template
5. Research & write pizzeria content
6. Gather/source images
7. Deploy to GitHub Pages

## Expected Output
- **URL**: https://fiddi-clawbot.github.io/barcelona-pizza/
- **Pages**: 1 home + 5 detail pages (6 total)
- **Assets**: ~15-20 images (3-4 per pizzeria)
- **Content**: ~500-800 words per pizzeria

---

**Estimated time:** 4-6 hours total
- Setup: 30 min
- Design/CSS: 1 hour
- Research: 1.5-2 hours
- Content creation: 1-1.5 hours
- Image sourcing: 30-45 min
- Deployment: 15 min
