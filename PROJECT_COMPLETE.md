## üöÄ What Happens When You Push

1. **You push to master branch**
2. **GitHub Actions workflow triggers** automatically
3. **Jekyll builds the site** (converts Markdown ‚Üí HTML)
4. **Built files deployed to gh-pages branch**
5. **GitHub Pages serves the new version**
6. **Site updates live** (within 30 seconds)

No manual build steps, no manual deployment. It's all automatic! ü§ñ

## üìù How to Add More Pizzerias

1. Create a new markdown file in `_pizzerias/name.md`
2. Add the frontmatter (name, award, cuisine, etc.)
3. Write the story and content
4. Add 3 images to `assets/images/`
5. Push to GitHub
6. **Done!** Site automatically rebuilds and deploys

Takes ~5 minutes per pizzeria.
